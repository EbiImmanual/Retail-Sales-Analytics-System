# Retail Sales Analytics and Reporting System

A Python and SQL based analytics project developed to analyze retail sales data and generate business insights using real-world datasets.

## Features
- Total sales and profit analysis
- Category-wise sales analysis
- Region-wise performance analysis
- Monthly sales trend analysis
- Top-selling product identification
- SQL-based analytical queries
- Automated business insight generation
- Data visualization using graphs

## Technologies Used
- Python
- Pandas
- SQL (SQLite)
- Matplotlib

## Visualizations
- Category-wise sales chart
- Region-wise sales chart
- Monthly sales trend graph

## Business Insights Generated
- Best Performing Region
- Best Performing Category
- Highest Selling Product
- Total Revenue
- Total Profit
